Instrucciones
El juego consiste en realizar nueve emparejamientos, que van apareciendo aleatoriamente, de forma adecuada. Cuando la relación se realiza de forma adecuada, la respuesta acertada desaparece del panel. Para completar el papel se dispone de un tiempo aleatorio comprendido entre 40 y 80 segundos. En el juego se indica el tiempo disponible, el número de errores cometidos y un panel donde se indica cuando las palabras no son correctas

El juego finaliza cuando se aciertan todas las palabras, cuando finaliza el tiempo o cuando se cometen más de diez fallos. Al finalizar se ofrece un cuadro resumen del resultado obtenido.

La aplicación permite la participación de una o dos personas. Cuando participen más de una persona, simplemente se debe abrir un panel diferente para cada una de ellas, cada uno aparecerá en un color diferente: naranja y azul. También permite jugar contra reloj o cualquier otra modalidad que se estime conveniente.

- Cuando un jugador pierda todos sus puntos, gana su contrincante.

Se puede llevar un control o evaluación de los resultados del juego mediante un formulario como el que se puede descargar al pulsar el siguiente botón.
