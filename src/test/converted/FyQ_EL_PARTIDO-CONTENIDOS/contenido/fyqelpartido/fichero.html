FyQ EL PARTIDO FICH
Con esta versión del juego, cualquier usuario puede crear y almacenar las preguntas que aparecerán en el juego ( Importante: para que el formulario funcione debe de utilizarse desde la página web de Internet o desde un servidor php). Los pasos a seguir son los siguientes:
  1. Pulsar el botón "Formulario de preguntas". Se abrirá una ventana en la que se deberán rellenar los campos de las preguntas. Después se debe pulsar el botón "Enviar".
  2. Se abrirá una ventana con los datos cumplimentados del formulario. Al final aparece el botón "DESCARGAR EL FICHERO GENERADO", al pulsarlo se abrirá una ventana para guardar el fichero generado donde se desee y con el nombre que se desee. Se trata de un fichero de texto editable donde se pueden corregir las erratas o fallos.
  3. Este fichero puede usarse en el ordenador del usuario si tiene el juego descargado en su ordenador local o enviarse por correo al coordinador de Proyecto Newton para su publicación en la web, para ello pulsar en el botón "Publicación de ficheros", donde se detalla la forma de hacerlo.
  4. Si se desea se puede consultar la lista de ficheros con baterías de preguntas publicadas en la web de Proyecto Newton, pulsando en el botón "Listado de ficheros publicados".
  5. Pulsar en la escena del juego "FyQ EL PARTIDO FICH", e introducir el nombre del fichero con cuyas preguntas se desea jugar en el sitio que se indica.
