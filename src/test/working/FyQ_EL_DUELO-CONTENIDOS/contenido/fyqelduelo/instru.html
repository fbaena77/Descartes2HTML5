FyQ EL DUELO. INTRUCCIONES

El formulario permite rellenar los datos necesario para cumplimentar cuatro preguntas para cada uno de los temas del juego FyQ EL DUELO. Hay cuatro bloques uno para cada tema del juego.

  1. Hay que poner el nombre de cada uno de los cuatro temas al empezar cada bloque de preguntas.
  2. El enunciado de cada la pregunta. Está dividido en seis campos, cada uno de ellos es una línea de la pregunta y se deben cumplimentar de izquierda a derecha.
  3. Para cada pregunta se deben dar tres posibles respuestas a la misma y se debe indicar cual es la correcta.
